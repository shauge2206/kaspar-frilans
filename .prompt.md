## Your design direction

**Fluid Modern** — soft, generous, premium feel. The site moves like water:
gentle gradient washes, thoughtful elevation, content that arrives on scroll
with spring-eased reveals, and a subtle, always-warm color story.

Signature move: a fluid gradient backdrop that slowly drifts (CSS-only or
canvas, performance-first) and a typographic hero where the lead headline
reveals word-by-word with spring physics, then settles. Section transitions
use scroll-linked animation (Framer Motion's `useScroll` / `useTransform`, or
the View Transitions API) so navigation feels continuous, not jumpy.

Direction-specific notes:
- **Layout**: generous whitespace, asymmetric grids, oversized type for
  headlines, slim metadata rows, soft 16–24px corner radius on cards. No
  hard borders — separation comes from elevation and color shifts.
- **Typography**: a contemporary display serif for headlines (e.g. Fraunces,
  Tiempos, Editorial New) paired with a clean grotesk for UI/body (e.g. Inter,
  Geist, Söhne). Variable fonts where possible. Tasteful kerning on display.
- **Palette**: warm paper-white base with a single deep ink and a soft
  accent gradient (e.g. apricot → coral, or lichen → teal). Avoid loud color.
  Dark mode is optional but if present, must feel intentional.
- **Motion**: spring physics (Framer Motion / motion-one). Stagger children
  on entry. Magnetic hover on primary CTA. Page transitions via View
  Transitions API or layout animations. ALL motion gated by reduced-motion.
- **Interactions**: smooth anchor scroll, scroll-linked progress on article
  pages, subtle parallax on hero imagery (small offsets only).
- **Components**: a refined article card (image, eyebrow, headline, meta), a
  publikasjoner row on `/om`, a values grid that reveals on scroll, a
  contact form with focus-ring micro-interactions.

You may install motion libraries you need: `motion` (motion-one) or
`framer-motion`. Keep the bundle reasonable.
