## Your design direction

**Precision Monochrome** – high-contrast, monochrome, exacting. The site reads
like Linear / Vercel / Stripe / Arc product pages: confident grids, tight
typography, near-zero ornamentation, and motion that's fast, snappy, and
purposeful. Photography is the only color.

Signature move: command-bar-style top navigation with a subtle blur and
hairline border, plus magnetic hover on primary actions. Page-to-page
transitions use the View Transitions API (or layout animations) so navigation
feels like one continuous surface. Article hero images animate with a sharp
mask reveal (left-to-right), text rows stagger up with short, fast easing.

Direction-specific notes:
- **Layout**: tight 12-column grid, hairline 1px dividers, deliberate
  whitespace, snap-aligned sections. Sticky article header with progress bar.
  No rounded corners larger than 6px; some elements use 0px (squared, sharp).
- **Typography**: a precise sans for everything (e.g. Inter Display, Geist,
  Söhne, Neue Haas Grotesk). Tabular figures for metadata. Tight tracking on
  large display weights. Optional mono (JetBrains Mono / Berkeley Mono) for
  metadata, captions, and tags only.
- **Palette**: pure monochrome – paper-white (#fafafa) and deep ink (#0a0a0a),
  with maybe one mid-grey. Photography supplies all color. Dark mode is
  optional but should feel like the *same* design, not a knockoff.
- **Motion**: short durations (140–240ms), tight cubic-beziers (Linear-ish
  `[0.16, 1, 0.3, 1]`), staggered entrances, magnetic hover, command-K palette
  with smooth open. Use Framer Motion / motion-one. ALL motion gated by
  reduced-motion.
- **Interactions**: keyboard-first feel – `⌘K` to open command palette
  (jump to articles, om, kontakt), `j/k` or arrow nav across `/saker` rows.
  Mouse stays first-class; keyboard is bonus.
- **Components**: a tabular `/saker` index with snap rows, a thin sticky nav
  with section dots on long pages, a cursor-aware image hover that pans 2-3px
  on mousemove, a contact form with inline status pill.

You may install motion libraries you need: `motion` (motion-one) or
`framer-motion`. Keep the bundle reasonable.
